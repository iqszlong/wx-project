# 安装依赖

```
pnpm i
```

# 运行项目

## 网页端

```
pnpm dev:h5
```

## 微信小程序

```
pnpm dev:weapp
```

# 打包

## 网页端

```
pnpm build:h5
```

# 微信小程序

```
build:weapp
```

# 运行以及打包路径

网页端：dist/h5

微信小程序：dist/weapp
